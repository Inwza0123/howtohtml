การใช้งาน Visual Studio เบื้องต้น



วิธีการเปิดใช้งานโปรแกรม Visual Studio


หลังจากที่ดาวน์โหลดและติดตั้ง Visual Studio แล้วให้เราเคลื่อนเม้าส์ไปที่รูปสัญลักษณ์ Windows หรือจะกดรูป Windows บนแป้นพิมก็ได้




เมื่อคลิกแล้วให้พิมพ์ตรง Type here to search ว่า Visual Studio แล้วจะขึ้นรูปโปรแกรมทันที เมื่อเจอแล้วให้ดับเบิ้ลคลิกเข้าไปก็จะเข้าสู่โปรแกรม Visual Studio



วิธีการเปิดไฟล์และบันทึกไฟล์

ถ้าจะสร้างไฟล์ใหม่ ให้คลิก File ในแถบเมนูด้านบน แล้วคลิก New File จากนั้นเริ่มพิมพ์โค้ด HTML ได้เลย
ถ้าจะเปิดไฟล์ที่มี ให้คลิก File ในแถบเมนูด้านบน แล้วคลิก Open File ไปยังไฟล์ HTML ที่จะเปิด แล้วคลิกเพื่อเลือก จากนั้นคลิก Open



วิธีการสร้างไฟล์ ให้กดคลิกขวาตรงพื้นที่ว่างแล้วเคลื่อนเม้าส์ไปกด New จะมีแถบอีกอันขึ้นแล้วให้เลือก Folder จากนั้นตั้งชื่อ Folder ของคุณ




เมื่อเลือกไฟล์มาแล้วเราสามารถพิมพ์โค้ดลงได้ทุกๆ เอกสาร HTML จะต้องขึ้นต้นด้วยการระบุประเภทเอกสารเสมอ
: <!DOCTYPE html> โค้ด HTML จริงๆ จะเริ่มต้นด้วย<html>และจบด้วย</html>ส่วนที่จะมองเห็นในเบราว์เซอร์จะอยู่ระหว่าง <body> และ </body>





หากต้องการบันทึกไฟล์ใน Visual Studio Code
คลิก File ในแถบเมนูด้านบน
คลิก Save as
พิมพ์ชื่อไฟล์ข้าง "File Name"
เลือก "HTML" ในเมนูที่ขยายลงมาข้าง "Save as type"
คลิก Save



คลิกตามภาพเพื่อรันข้อมูลเมื่อคลิกแล้วข้อมูลจะปรากฏบน Chrome เพื่อตรวจสอบว่าโค้ดที่เขียนลงไปนั้นถูกต้องหรือใช้งานได้หรือไม่ หรือจะพิม 127.0.0.1:5500/ชิ่อไฟล์.htmlก็ได้
